About AWS Decision Assistant

There are many database solutions available on AWS.
Choosing the right tool is not always easy, and it can be vital to the success of your project.

AWS Decission Assistant is a sipmle quiz-like project
created as a Jinja exercise and consolidation of knowledge about AWS database services,
which can be helpful in choosing the right tool from the AWS offer.
