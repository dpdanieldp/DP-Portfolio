Book Collection

  • About Book Collection

    Application allows you to create db with books info.

    You can add books yourself or import books from Google Book by criteria.

    It is possible to search in your book collection, edit end delete books.

    Application serves also REST API queries - you can search books in your collection by some values.

    Working REST API query pattern:

    • X- your search
    • title- search in tiles
    • author- search in authors
    • language- search in languages
    • fromdate- search FROM this publication date (date format: YYYY-mm-dd)
    • todate- search TO this publication date (date format: YYYY-mm-dd)

    API URL:

    http://gbapidp.us-east-2.elasticbeanstalk.com/book-collection/api/query?title=X&author=X&language=X&fromdate=X&todate=X
